# 🍽 Taste Lounge

**Taste Lounge** is a production-ready full-stack restaurant ordering web application featuring secure user authentication, real-time order tracking, and seamless Paystack payment integration.

---

##  Live Demo
🔗 https://taste-lounge-3075.web.app

---

##  Overview

Taste Lounge is a modern digital restaurant platform that allows users to browse meals, add items to cart, place orders, and complete secure online payments. The system is built with scalability, performance, and user experience in mind.

---

##  Features

- 🔐 Secure user authentication (Firebase Authentication)
- 🛒 Add-to-cart and order placement functionality
- 💳 Secure Paystack payment integration
- 📦 Real-time order tracking via Firestore
- 📱 Fully responsive and mobile-friendly UI
- ⚡ Optimized performance and clean interface design
- 🔒 Protected routes for authenticated users

---

##  Tech Stack

### Frontend
- HTML5
- CSS3

### Backend & Services
- Firebase Authentication
- Firebase Firestore (Database)
- Paystack Payment API
- Firebase Hosting (Deployment)

---

##  Architecture Overview

Taste Lounge follows a modern full-stack architecture:

- The frontend handles UI interactions, cart logic, and user actions.
- Firebase Authentication manages secure user login and sessions.
- Firestore stores order data and enables real-time updates.
- Paystack API securely processes online payments.
- Firebase Hosting serves the production build.

---

##  Security Implementation

- Secure authentication via Firebase
- Encrypted user sessions
- Protected routes for logged-in users
- Secure third-party payment processing (Paystack)
- No sensitive card information stored within the application

---

##  Deployment

The application is deployed using Firebase Hosting for high availability and performance.
